CSS Align - Horizontal & Vertical Align

Horizontally

* Block Element

To center a block element, use: margin: auto;

Note: Must set the width property

Example

* Inline Element

To center a text inside an element, use: text-align: center;

Example

* Center an Image

To center an image, update the display element to block and set the margin: margin: auto, display: block;

Left & Right Align

* Using position

You can also use: position: absolute; and update right/left properties.

Note: Using absolute position will remove the element from the normal flow.

* Using position & transform

  1. Set the parent div position to relative
  2. Set the child div position to absolute
top: 50%; left: 50%; transform: translate(-50%, -50%);
position: absolute; top: 50%; left: 0%;

* Using flexbox

To center horizontally & vertically using flex, set justify-content and align items to center:

display: flex; justify-content: center; align-items: center;
